# 🎬 Movie Explorer Project

A modern and responsive **Movie Explorer web application** built using **HTML, CSS, and JavaScript**.  
This project allows users to browse movies, search for specific titles, and view detailed information on a separate page.


## 🌐 Live Demo
🔗 https://filmy-explorer.netlify.app/ 


## 📌 Overview
This application fetches and displays movie data in a clean and responsive card layout.  
Users can search for movies using the search bar and click on any movie card to view detailed information on a separate page.

The project is designed to provide a smooth and interactive user experience with fast navigation and simple UI.


## ✨ Core Features

🎬 Display movies in responsive card layout  
🔍 Real-time search functionality for movies  
📄 Click on movie card to view detailed page  
📊 Separate movie details page with full information  
⚡ Fast and smooth UI interaction  
📱 Fully responsive design (mobile + desktop)  


## 🛠️ Technology Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- REST API (Movie Data Source)


## ⚙️ Project Workflow

1. Fetch movie data from API  
2. Display movies in card format  
3. User searches movie via search bar  
4. Filter results dynamically  
5. Click movie card → open detailed page  
6. Show full movie details on new page
